## Project Overview

**Math Tutor** is a Socratic AI tutor that guides students through math problems step by step, rather than giving full solutions upfront.

**Core problem it solves:**
Standard AI gives walls of text. Practice problems generated by AI are too easy and too generic to prepare for real exams. This project fixes both.

**Planned features:**
- [x] Multi-turn conversation
- [x] Socratic system prompt
- [ ] Image / screenshot input
- [ ] PDF input
- [ ] RAG-based question bank from past exams
- [ ] Generate new questions matched to real exam difficulty
- [ ] Mistake tracker (SQL)
- [ ] Web interface (Streamlit)

**Tech stack:** Python · DeepSeek API · LangChain · ChromaDB · SQLite · Streamlit
